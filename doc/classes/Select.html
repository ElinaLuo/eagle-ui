    Select

    下拉选择框组件
    支持在input框中输入文字或者点击下拉菜单中的选项进行查询,同时支持上下方向键进行选择。 主要属性接口：

    • defaultChecked:默认显示的选项即选中的选项
      如： defaultChecked="上海"
    • getValueCallback:获取元素之后的回调方法，主要将选中值传递给父级元素
      如：getValueCallback={::this.getValue}
    • placeholder:提示框的内容
      如：placeholder="请选择"
